# اختبار Git: التحدي المتوسط

1️⃣#  أنشئ فرعًا باسم feature-z وأضف ملفًا باسم idea.txt يحتوي على "الفكرة الأولى". ثم قم بعمل Commit.

2️⃣#  أضف سطرًا ثانيًا "الفكرة الثانية" إلى idea.txt وقم بعمل Commit مرة أخرى.

3️⃣#  عد إلى الفرع main وادمج الفرع feature-z.

4️⃣#  قم بإزالة آخر Commit من main ولكن احتفظ بالتغييرات الموجودة في الملف.
